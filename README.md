# Node.js Event Loop Visualizer

An interactive **Node.js event loop visualizer** built with **Next.js**, **React**, and **Tailwind CSS**. Explore how **setTimeout**, **setImmediate**, **process.nextTick**, and **Promise microtasks** run across **libuv event loop phases** with a live, step-by-step execution timeline.

## Why this project

Understanding the Node.js event loop is critical for debugging async behavior and optimizing performance. This visualizer helps you learn the **JavaScript event loop**, **microtask queue**, **nextTick queue**, and **timers/check/close** phases through interactive examples.

## Features

- Interactive **Node.js event loop visualization** with queues and phases
- Examples covering **timers vs immediate**, **nextTick priority**, and **microtasks**
- Live **console output timeline** and **call stack** animation
- Adjustable execution speed for better learning
- Built with **Next.js App Router** and **Tailwind CSS**

## Tech stack

- **Next.js** (App Router)
- **React**
- **TypeScript**
- **Tailwind CSS**
- **lucide-react** icons
